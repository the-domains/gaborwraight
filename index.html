Jide

Remix OS is a x86 Based Android Operating System built for PC. 

The guys from jide.com managed to give Android a Desktop look & feel by adding a file manager, a taskbar, right-click, notification and so much more. You can make a bootable USB stick which you can carry with you to boot whatever computer you want into your Remix OS. If you want you can of course also install it natively on you machine.
